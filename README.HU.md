## ⚡ Hogyan használd

1. Válassz promptot a `/prompts/hu/` vagy `/prompts/en/` mappából
2. Töltsd ki a `[VÁLTOZÓK]` részt
3. Modulokat aktiválj `+` / deaktiválj `-`
4. Illeszd be bármely LLM-be — Claude, Gemini, ChatGPT, Gemma
